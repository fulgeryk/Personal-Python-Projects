# Image Watermarking Desktop App

A desktop application built with Python, Tkinter, and Pillow that allows users to upload an image and add a customizable watermark.  
The app provides a simple graphical interface to preview and save watermarked images.

---

## Features

- Upload an image from your computer
- Add text watermark
- Adjustable watermark opacity
- Preview inside the application
- Save the processed image

---

## Tech Stack

- Python 3
- Tkinter (GUI)
- Pillow (image processing)

---

## Project Structure

- main.py # Entry point
- ui.py # User interface logic
- watermark_engine.py # Image processing engine


---

## Learning Goals

This project focuses on:

- GUI development with Tkinter
- Image manipulation using Pillow
- Separation of UI and logic (clean architecture)
- Building real-world desktop applications
