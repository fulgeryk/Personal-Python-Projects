# Personal Portfolio Website

A responsive personal portfolio website built with Bootstrap, designed to showcase projects, skills, and development direction in a clean and structured way.

This project serves as a foundation portfolio that will evolve over time as new projects are added.

---

## Features

- One-page responsive layout
- Project showcase with GitHub links
- Clean portfolio card structure
- About section describing development focus
- Contact section with social links
- Modern UI built with Bootstrap
- Scalable structure for future backend integration

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome

---

## Project Structure

- Hero section introducing developer identity
- Projects gallery with modal previews
- About section with development philosophy
- Contact section for external links
- Footer with branding

---

## Running Locally

Clone the repository and open the main file in a browser.

No backend or build tools are required.

---

## Future Improvements

- Flask backend integration
- Functional contact form
- Live deployment
- Blog section
- Theme customization
- Analytics integration

## Purpose

This portfolio is part of a continuous learning journey focused on Python development, automation, and full-stack growth.

It is intentionally designed to grow alongside future projects.
